## 1. Как установить?

### 1.1 Что нужно установить
- **Node.js** https://nodejs.org/en

### 1.2 Установка зависимостей

1. Клонируйте репозиторий:  
   ```bash
   git clone https://github.com/kekhar/shurik.git
   ```

2. Установите пакеты:
	```bash
   npm install
   ```

## 2. Как запустить?
1. Запуск в режиме разработки
	```bash
	npm run dev
   ```
